This project analyzes Indian airline ticket pricing data using Power BI to understand how factors like airline, class type, duration, stops, and timing influence fares.
Interactive dashboards were created to compare economy and business class prices and identify pricing trends across different routes and airlines.
Tools Used: Power BI, CSV datasets, Data Cleaning & Visualization
Details of the Dataset for this data is obtained from kaggle
Link - https://www.kaggle.com/datasets/dhirajbembade/indian-airlines-ticket-price-analysis?resource=download
